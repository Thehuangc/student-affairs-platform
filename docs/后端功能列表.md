# 学生综合事务中台 - 后端功能列表

## 1. 技术架构

### 1.1 技术栈
- **运行环境**: Node.js 20.x
- **Web框架**: Express.js
- **数据库**: MySQL 8.0
- **ORM**: mysql2/promise（原生SQL）
- **认证**: JWT (jsonwebtoken)
- **密码加密**: bcryptjs

### 1.2 项目结构
```
backend/
├── server.js          # 主服务文件
├── package.json       # 依赖配置
└── init-database.sql  # 数据库初始化脚本
```

## 3. 核心业务功能

### 3.1 认证模块
- 用户登录（JWT Token生成）
- 用户注册
- Token刷新
- 用户登出

### 3.2 用户管理
- 用户CRUD操作
- 密码重置
- 角色管理CRUD

### 3.3 志愿服务
- 活动管理（CRUD、状态流转）
- 活动报名（人数校验、重复报名校验）
- 签到签退（签到码验证、时长自动计算）
- 志愿者时长统计

### 3.4 团员发展
- 入团申请（提交、审核）
- 政审备案（新增、审核）
- 电子档案生成

### 3.5 勤工助学
- 岗位管理（CRUD）
- 应聘申请（提交、审核）
- 考勤管理（记录、审核）
- 薪资管理（生成、确认、发放）
